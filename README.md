##　AdapterDemo

这个项目是由`ListView`的升级版`RecyclerView`构建出来的一个列表项，里面数据我没有进行加入，然后用一组数据生成一万次构成了一个列表，利用适配器进行数据的传输，实现了`Recycler View`的点击事件，可以用来删除你点击的`item`。
